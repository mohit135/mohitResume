Conceptohub - everything at your fingertips...


Conceptohub is an online platform where students get the class notes according to their college curriculam at doorstep.

 

Problem Statement: The requirement was to design and build a platform where students could get the following things.

  • Class Notes according to their college.
  • Class notes should be present Semeter-wise followed by the subjects.
  • Previous Years papers
  • If College is not listed, then there should be generic class notes available easily.
  • Intersting facts page where students can get the intersting articles.
  • A forum to discuss things.
  • Should be able to like and share the artciles & page on facebook.
  • Students can login and make their user profiles.
  • Client: Conceptohub Solutions
  • Date: January 2014
  • Service: Web Development

Gehna Ville - Jewelry that defines you...


GehnaVille is e-retail platform that has exclusive collection of fashion jewellery and accessories.

 

Problem Statement: The requirement was to design and build a full-flegded website which has following features:

  • It should include Content Management System, where admin can easily upload the products and maintain portal.
  • User can easily find the best design according to their suitablity.
  • User should be able to filter on the basis of different categories and sort it.
  • User should be able to order it easily via online payment.
  • Intersting facts page where students can get the intersting articles.
  • User can easily track the shipping and check its status.
  • Should be able to like and share the artciles & page on facebook.
  • It should have facility to write reviews.
  • Client: Gehna Ville
  • Date: August 2017
  • Service: Web Development
